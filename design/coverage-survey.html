国家与路径覆盖草案

面向 Phase 1+ 路径扩张规划 决定覆盖范围、不决定具体数值 v0.2 · 加入三阶段小结

这份文档是什么：一份"覆盖范围扫图"，用来决定未来路径卡片列表里应该出现哪些路径（无论是已覆盖还是未覆盖）。 不是什么：这不是一手源数据——此处的路径名、大致门槛、时长全部未经本项目的一手源核验，只用于规划与讨论。任何一条被列的路径，在真正进入产品之前，都必须按 RULES § 2.1 重新从官方源抓取、快照、审阅。 评级含义：是"对本项目核心画像（中文母语技术/知识工作者单干）的契合度 + 作为对照组的教育价值"综合判断，非"路径本身的优劣"。 v0.2 新增：每国下方的「工签 / 永居 / 入籍」三阶段小结，是对照 sitemap v0.3.1 的路径卡片字段结构做的首次压测——为了看清哪些信息属于"路径级"、哪些属于"国家级"、哪些得做成"混合"。同样非一手源，数字仅用于判断字段归属与工作量，不能作为用户可见内容。

Tier A主流目的地 · 必覆盖

面向"中文母语技术/知识工作者"的七个基石国家。路径覆盖丰富、一手源稳定、中文用户存量大。

日本 JP · jp

Phase 0 唯一落地国。地理/文化近、技能签证规则条文化程度高、中文用户关注度最高。

工签
通道丰富：积分制 HSP（★★★）+ 常规工签 技人国（★★★）+ 创业 経営・管理。规则条文化程度在亚洲最高。路径级
永居
HSP 70 分 → 3 年；HSP 80 分 → 1 年；普通签 → 10 年。强路径耦合——永居年限是 HSP 的真正卖点。路径级
入籍
归化 5 年 + 日语 + 放弃原国籍（日本不允许双重国籍）。对中国用户＝同时失去中国国籍。一套法条，所有路径共用。国家级
评级路径简介 / 适用性
★★★高度専門職 1 号ロ · 技術系Highly Skilled Professional · Type I (b)积分制，学历 + 年薪 + 年龄 + 日语 + 研究成绩等。70 分准入、80 分加速。3 年 / 1 年可申永居。Phase 0 覆盖的唯一路径。
★★★高度専門職 1 号イ / ハHSP · Type I (a) / (c)学术研究 / 经营管理。同属积分制。与ロ同引擎，字段集共享，扩张成本低。
★★★技術・人文知識・国際業務Engineer / Specialist in Humanities / Intl. Services最主流的工作签证。不走积分、走职位/学历/契约匹配。覆盖绝大多数日企就职场景，是 HSP 的替代/预备路径。
★★特定活動 46 号Designated Activities · No.46本科以上 + N1 对部分跨专业就职开放。适合日语已达线但专业不匹配的人。
★★経営・管理Business Manager开公司投资签。~500 万日元资本 + 实体办公 + 两名雇员要求。单干创业常见路径。
★J-Find / J-SkipFuture Creation Individual · 未来创造人材2023 新设。Top 大学毕业生 2 年求职签 / 高年收者快速通道。窗口小但有战术价值。
★特定技能 1 号 / 2 号Specified Skilled Worker制造/建筑/护理等蓝领。不匹配核心画像，但应列以显示"我们选择不覆盖的原因"。
★配偶者签证 · 日本人の配偶者等Spouse of Japanese National家庭路径代表。列入以支持"我主路径不在这里但想了解"。

美国 US · us

中文用户关注度极高。系统的抽签 + 排期特性使它与日本的打分制完全不同——这是"路径卡片"数据结构必须兼容抽签制 / 排期制的主要原因。

工签
抽签 + 排期双重机制：H-1B 抽签、O-1/L-1 无抽签、EB-1/2/3 靠雇主排队。规则层级多（联邦法、USCIS 口径、月更排期表）。路径级
永居
绿卡本身就是永居。中国大陆出生者 EB-2/3 排期以十年计——这条在官方排期表上月更。路径级
入籍
绿卡 5 年 → 公民（婚姻路径 3 年）。允许双重国籍（但中国单方面不承认）。入籍考试 + 宣誓。国家级
评级路径简介 / 适用性
★★★H-1B · 专业人士工作签H-1B Specialty Occupation年度抽签。本科对口专业 + 雇主担保。中国大陆出生者后续 EB 排期极长是核心难点。
★★★O-1 · 杰出人才O-1 Extraordinary Ability学术/产业/艺术成就。无抽签、可续。近年技术从业者讨论度极高。
★★★EB-1A / EB-1B · 杰出人才/杰出研究Employment-Based First Preference直接指向绿卡。对中国大陆出生者有排期，但远短于 EB-2/3。
★★★EB-2 NIW · 国家利益豁免EB-2 National Interest Waiver硕士 + 国家利益论证。自主申请、不需雇主。中国大陆出生者排期是最大痛点。
★★L-1A / L-1B · 跨国公司派遣Intracompany Transferee1 年以上海外母公司任职转调美国。L-1A 指向 EB-1C 绿卡。
★★O-1 → EB-1A 组合路径非独立路径，但事实标准。应作为"路径组合 / 战术"的首个案例。
★★E-2 · 条约投资者E-2 Treaty Investor中国大陆护照不在条约国列表，需先持第三国护照。列入以显示"护照硬约束"是个独立维度。
★EB-5 · 投资移民EB-5 Investor80 万/105 万美元投资。资金门槛不匹配主画像，但作为"钱换时间"的对照必须列。
★F-1 → OPT → H-1B 学生路径Student Track低龄/在学用户的主路径。Phase 2 应覆盖，现阶段先列。
★DV 抽签Diversity Visa Lottery中国大陆出生者多年不在资格国。列入以展示"出生地硬约束"。

加拿大 CA · ca
