# relocate

一个业余维护的"移民 / 移居路径"参考网站。

仓库名 `relocate` 是内部代号，不是最终网站名。

---

## 是什么

把各国对**外国专业人才**的签证、永居、入籍规则，以及与之配套的家属签、
常见税负、生活成本、时间线整理到一起，方便中文读者**了解和对比**。

写法偏编辑风：每条信息尽量附官方来源链接；能存原件的就在 `data/archive/` 存一份；
不做炫酷 dashboard，不做"AI 移民顾问"聊天框。

## 不是什么

- **不是权威信息** — 真正做决定前请到对应国家的政府官网亲自核对
- **不是决策工具** — 不说"推荐哪条路"，也不替你比较"哪个更适合你"
- **不是商业产品** — 永远不会是。不接广告、不接赞助、不接中介、不做付费墙
- **不是移民中介** — 不做代办、陪跑、保过、一对一咨询
- **不对任何人的移民结果负责** — 参考它是读者自己的选择，后果读者自担

## 信息质量

尽力而为，不做硬保证。详见 [RULES.md](RULES.md) § 2。

- 有具体数字或规则的陈述，尽量附官方源链接
- 允许引用律所 / 博客 / 论坛作为"二手"线索，但会显式标注
- LLM 辅助整理文字可以；LLM 生成事实性数字不行
- 信息过时是常态——发现了就改，改完留 commit 记录

## 目录结构

```
relocate/
├── RULES.md          # 项目基调原则（非硬章程）
├── README.md         # 本文件
├── HANDOFF.md        # 接手说明（给未来的自己 / 任何一起维护的人）
├── SOURCES.md        # 已引用过的一手源清单（累进式，不强制前置）
├── site/             # 可部署的静态站：首页 + 路径详情页
├── data/             # 引用过的官方文件本地归档
├── design/           # 早期视觉 / 结构探索（非可部署内容）
├── docs/             # 工序与协作笔记
└── archive/          # 试错留痕，永不删除
```

技术栈与部署说明见 [docs/engineering.md](docs/engineering.md)。
