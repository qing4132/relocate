## 范围边界（和 eb1a / l1b 的关系）

EB-1B 与 EB-1A / EB-1C 同属 EB-1 第一优先，共用大量下游文件：
- **所有 IV 类共用**（INA §203 / §245 / §202 / §316 / §320 · CSPA · CCA · AC21 · 8 CFR 204.5 框架 · 245 · 22 CFR 42.12 · DOS Visa Bulletin · DOL FLAG · I-140 / I-907 / I-485 / I-765 / I-131 / I-693 / DS-260 / N-400 / N-600 …）→ 一律沿用 `../l1b/`，本目录不重复
- **EB-1 家族共页**（eb1_overview.html）→ 已在本目录保存一份；eb1a 目录亦独立保存，互不干扰
- **EB-1B 独有**（Vol 6 Part F Ch 3 政策手册 · 6 条证据标准 · permanent research position 定义）→ 本目录 `uscis/`
