# data/archive/us/eb5/ · 官方原件归档

> 范围：USCIS EB-5「投资移民」Immigrant Investor Program
> 法源 INA §203(b)(5) · 细则 8 CFR 204.6 + 216.6 · PubL 117-103 Div BB（EB-5 Reform and Integrity Act of 2022，简称 RIA）

## 范围边界（和 l1b 的关系）

EB-5 是 IV 类移民，但细则、表格、审查标准和其他 EB 类差异极大，**独有**文件较多：
- 共用件（INA §245 AOS 总则 / INA §316/319/320 入籍 / CSPA / 22 CFR 42.12 / Visa Bulletin / I-485 / I-765 / I-131 / I-693 / DS-260 / N-400 / N-600）→ 沿用 `../l1b/`，本目录不重复
- **EB-5 独有**细则（8 CFR 204.6 申请 · 216.6 条件解除）· 法条（RIA 2022 PubL 117-103）· 表格（I-526 · I-526E · I-829）· 政策手册（Vol 6 Part G）→ 本目录

## 编辑原则

- 不在本目录重复保存 l1b/ 已有的共享文件
