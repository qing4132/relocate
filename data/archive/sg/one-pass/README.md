# data/archive/sg/one-pass/ · 新加坡 ONE Pass（Overseas Networks & Expertise Pass）

面向 [`site/paths/sg-one-pass.html`](../../../site/paths/sg-one-pass.html) 的官方原件快照。

**范围边界**：本目录放 ONE Pass 相关的 MOM / ICA / IRAS / AGC 指南与法源。与新加坡其他准证（EP / Tech.Pass）相邻的参考链接也一起收下，因为它们是 ONE Pass 的下位 / 替代类别；永居（PR）、公民、税务居民判定属于所有外籍工作者共性话题，同样收入本目录，便于首次路径就有完整链路。

## 下载

```bash
./download.sh
```

- 幂等（已存在跳过 `SKIP`）
- 日志写入 `_download.log`（根 `.gitignore` 已 ignore `*.log`）
