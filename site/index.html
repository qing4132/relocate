关于搬去别的国家生活，我们自己整理的一份笔记

一个业余维护、非盈利、不对任何人负责的中文移民 / 移居路径参考网站。

这是什么：把几个国家对外国专业人才的工作签证 → 永居 → 入籍规则，以及与之配套的家属签、常见税负与生活成本整理到一起，方便中文读者了解和对比。

不是什么：不是权威信息、不是决策工具、不是商业产品、不是移民中介。

请这样用：把它当作"从哪里开始查"的索引，而不是结论。真的要做决定前，请到对应国家的政府官网亲自核对最新规则，必要时咨询执照律师。

这个项目不做什么

  • 不做商业化 — 永远不接广告、不接赞助、不接中介、不做付费墙、不做订阅。
  • 不做代办 / 陪跑 / 咨询 — 不为任何人的移民结果负责。
  • 不做 AI 移民顾问 — 不做聊天式"推荐路径"产品。
  • 不替你选 — 呈现信息、列出差异，停在那里；选哪条、去不去，是你自己的事。
  • 不做增长 — 不推广、不拉新、不做"邀请返利"这类套路。自用 + 顺手公开。

信息质量承诺

尽力而为，不做硬保证：

  • 有具体数字或规则的陈述，尽量附官方来源链接。
  • 允许引用律所 / 博客 / 论坛作为"二手线索"，会在页面上显式标注"二手"。
  • LLM 辅助整理文字可以，LLM 生成事实性数字不行。这是本项目的硬底线。
  • 信息过时是常态——发现了就改，改的时候 commit 记录说清"从哪查出来的、改成了什么"。
  • 本站不追踪访客、不用 cookie、不跑分析脚本。
