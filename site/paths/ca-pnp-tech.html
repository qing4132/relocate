§ 0全景 · PNP 是什么 · Enhanced vs Base · 与 EE 的关系

PNP（Provincial Nominee Program）是 IRPA §87 之下的联邦 + 省双层程序：联邦把"经济类移民"配额按"人口 + 经济需求"分配给各省；各省自己设计提名标准（如紧缺职业 / 工资门槛 / 在省居住等）；申请人拿到省提名（Provincial Nomination Certificate）后 → 联邦 IRCC 仍是 PR 的最终发签机构。

[ 联邦 / 省 / 申请人三方关系 ] IRCC（联邦 · IRPA §87） │ 分配年度配额给各省 ▼ 13 个省 / 地区各自的 PNP（13 + 1 含 Yukon） │ 自定 streams / NOC / 门槛 / 配额 ▼ 申请人 提交 Expression of Interest（EOI 或类似） │ ▼ ┌────────────────────┴───────────────────┐ │ │ ▼ Enhanced PNP ▼ Base PNP 走 Express Entry pool 不走 Express Entry 省提名 +600 CRS → 几乎必拿 ITA 直接递 IRCC paper-based PR PR 处理 6 月 · 配偶子女同批 LPR PR 处理 12-22 月 · 较慢 │ ▼ COPR · 登陆 PR │ 3 年 / 5 年 居住 ▼ 入籍（同 ca-gts § 4）

0.3PNP 在 IRCC 配额博弈中的位置

2024 年 PNP 总配额砍半 · 各省随之缩水

2024 年下半年，IRCC 把 PNP 年度总配额从 ~ 110,000 砍到 ~ 55,000；2025–2027 年保持低位。这对申请人意味着：

  • 各省 draw 频率与每次邀请数显著下降
  • OINP / BC PNP / AAIP 的 cutoff 普遍上调（OINP Tech Draw 历史最低 CRS 421，2024 下半年起多次升到 460+）
  • 部分原本"几乎必中"的省提名通道（如曼省、萨省 EOI 排队）变成"分数游戏"
  • PNP 时间窗口越来越宝贵——"先工签后省提名"路径比"裸 PNP"更可控

这条政治背景是 2025 起规划 PNP 路径时无法忽视的现实约束。本页以下所讲的"凑分玩法 / 通道选择 / 风险点"全部假设这一基本面。

2.1OINP · Tech Draws 详细

§2.1 · ontario.ca · OINP Tech Draws
OINP · Human Capital Priorities Tech Draws

独特机制：OINP Tech Draw 是不需要 Job offer的 PNP 通道 —— 这点在 13 个省的 PNP 里非常少见。条件：

  • 已在联邦 Express Entry pool 持有有效档案
  • EE 档案中 NOC 在 10 个 Tech NOC 之一：
    20012（CIS managers）· 21211（Data scientists）· 21223（Database analysts）· 21230（Computer systems developers）· 21231（Software engineers）· 21232（Software developers）· 21233（Web designers）· 21234（Web developers）· 21311（Computer engineers, non-software）· 21468（Computer and network operators · 部分包含）
  • CRS ≥ OINP Tech Draw cutoff（2024–25 浮动 425–460）
  • 过去 5 年内累计 1 年 NOC 相关工作经验
  • "打算居住在 Ontario"（陈述）
  • 英语 / 法语 CLB 7（NOC 0/1）或 CLB 5（NOC 2/3）

流程：OINP 在 EE pool 内主动发 Notification of Interest（NOI）给符合条件的候选人 → 申请人接受 NOI 后 14 天内向 OINP 递省提名申请 → OINP 审 60–90 天 → 提名信 → EE 系统自动加 +600 CRS → 几乎必拿 ITA → 走 IRCC 完成 PR。

价值："裸 EE 进池 + 等 NOI" 是中国大陆专业人才不需要先去加工作即可拿 PR 的少数路径之一 —— 但等 NOI 的随机性高（2024 全年仅个位数次 Tech 专项 draw）。

2.2AAIP · Alberta Express Entry Stream 详细

§2.2 · alberta.ca · AAIP Express Entry Stream
AAIP · Alberta Express Entry Stream

独特机制：AAIP EE Stream 也是不需要 Job offer的（但有则显著加分）· 由 AAIP 主动从 EE pool 邀请。条件：

  • 已在 EE pool 持有效档案（CEC / FSW / FST 均可）
  • NOC TEER 0/1/2/3（不限 Tech 专项 · 全行业 · 但 Tech / 工程 / 医疗加分高）
  • CRS ≥ 当期 cutoff（2025 浮动 300–400 · 比 OINP 低一档）
  • "intention to reside in Alberta"（陈述）
  • 有 AB 工作 / AB 学历 / 配偶在 AB 等可加分

流程：同 OINP · AAIP 主动发 NOI → 申请人 30 天内接受并递 AAIP 申请 → AAIP 审 6 个月 → 省提名 → EE +600 → ITA → IRCC PR。

价值：CRS cutoff 比 OINP / BC PNP 低 30–80 分 · 适合 CRS 自评 380-430 区间的申请人。

三大省 Enhanced 通道的流程框架高度一致——差异在于"省提名前是否需要 Job offer"与"省提名审理时长"。

3.2BC PNP / AAIP 流程的差异点

  • BC PNP：不是 OINP "等 NOI" 模式 · 而是主动注册 BC PNP profile（独立于 EE）→ 提交 EOI → BC PNP 每周二 / 四发 Invitation to Apply（ITA-BC，不是联邦 ITA）→ 拿到 BC ITA 后 30 天递省提名 → 60–120 天审 → 拿到 Nomination → 再回 EE 加分。关键差异：BC PNP 流程"前置"在 BC 而非 IRCC EE。
  • AAIP Express Entry：同 OINP "等 NOI" 模式 · 但 NOI cutoff 更低 · 邀请频率类似。审理 6 个月（vs OINP 60-90 天 · vs BC 60-120 天 · AAIP 实务上最慢）。

3.3家属同步

  • 配偶 / 同居伴侣 / 22 岁以下未婚子女都作为 PR 申请的派生家属，与主申同批审理 · 同批 COPR · 同期登陆
  • 已在加持工签的配偶（如 OWP / SP）继续保持工签身份直至 PR 落地
  • 境外配偶登陆 PR 一次即生效，之后不再需要工签转换

3.4"等省提名期间"的工签桥梁

PNP-Work Permit Support Letter · 等待期工签

若拿到省提名但 PR 还未批，且急需在该省入境工作，可向省 PNP 申请 Support Letter → 凭此向 IRCC 申 LMIA-exempt 工签（雇主代码 A75 / C50 等）· 通常 1–3 年 · 与 GTS / ICT 工签并列同一类。

这对"国内已没工作 + 拿到 OINP 提名 + 等 IRCC PR 6 月"的家庭有用 —— 直接拿 Support 工签全家先过去，PR 落地后转。

§ 4PR 维持 + 入籍 · 引用 ca-gts § 3–§ 4

拿到 PR 之后的剧本与 ca-gts / ca-ict 完全一致——本节用 .reuse 直接引，不重复。

引用 · ca-gts § 3.4 PR 维持：PR 卡有效期 5 年 · 必须在 5 年窗口内在加拿大物理存在 ≥ 730 天（2 年）· 长期返中工作 / 加方雇主派驻 / PR 配偶随加公民海外生活等可计天数。详见 ca-gts § 3.4。
PNP 专属补充：多省 PNP（OINP / BC / AAIP / SINP / MPNP 等）在提名时要求申请人陈述"打算在该省长期居住"——这是道德义务而非法律强制，但若 PR 拿到后立即跨省（如 OINP 后搬去 BC）下次续 PR 卡时可能被询问 · 严重者 OINP 可向 IRCC 报告"misrepresentation"。建议至少在该省居住 1–2 年再考虑迁移。
引用 · ca-gts § 4 入籍：Citizenship Act §5 五条门槛——18 岁 / 5 年内累计 1,095 天物理存在 / 5 年内 3 次报税 / 入籍考试 / 18–54 岁 CLB 4 语言 / 无重大犯罪 / 宣誓。处理 12–18 个月。详见 ca-gts § 4 全段。
中国国籍法 §9 同步触发：加入加籍即自动失中国籍 · 注销中国护照 / 户籍。

4.1PNP 路径入籍时间线推算

  • Y0 拿到 EE 档案 + 等 / 申省提名
  • Y0.5 拿到省提名（OINP 60–90 天 · BC 60–120 天 · AAIP 6 月）
  • Y1 EE +600 CRS · 拿 ITA · 递 e-APR
  • Y1.5–Y2 PR 批 · COPR · 全家登陆
  • Y2–Y5 持 PR 累 3 年物理存在
  • Y5 申入籍（处理 12–18 月）
  • Y6 入籍仪式 · 拿加拿大护照

总时长 ~ 6 年 · 与 ca-gts（GTS 工签 → CEC → PR）/ ca-ict 节奏相近。"裸 PNP" 无工签前置时整个流程都在 IRCC 文件审核里 · 比 GTS 那种"先 2 周入境工作"路径心理压力大（住房 / 落地 / 子女学校的时间都得等 COPR）。

§ 5家庭玩法 · PNP + 工签 / 双 EE / Support Letter

PNP 路径的家庭维度比 GTS / ICT 简单——一份省提名申请 + 一份 PR 申请就覆盖全家 · 没有"派生工签 / 家庭双线"的额外复杂性。但有一些独特玩法值得说。

5.1主线 · 主申 PNP + 全家同批 PR

最常见 · 一份省提名 → 一份 PR → 全家登陆

主申拿到省提名 → 递 PR 时配偶 + 22 岁以下未婚子女作为派生家属同批申请 · 一次性 COPR 全家 PR · 一次登陆全家身份生效。

这是 PNP 路径相对 GTS / ICT 最大的简化—— 没有 "配偶 OWP / 子女 SP" 这种 "工签时代的派生" 步骤，直接跳到 PR。

5.2双线 · 主申 PNP + 配偶独立 EE

两人都有 EE 档案 · 谁先拿到 ITA 谁主申

若夫妻双方都各自满足 EE 资格（CEC 通常需先有 1 年加拿大经验 · FSW 海外 1 年即可），可双方各自建 EE 档案：

  • 主申方申 PNP · 拿到省提名 +600 几乎必中 ITA
  • 配偶方独立 EE pool 等 NOI / 一般 draw cutoff（无 +600 · 须靠自身 CRS）
  • 主申拿到 ITA → 配偶作为派生家属一同入 PR · 无需自己单独走 PR

价值：配偶的独立 EE 档案是 plan B —— 若主申 PNP 申请失败（如 NOC 不在清单 / 工资不达 / OINP 改革等），配偶的 EE 档案可直接顶上（自身 CRS 高时也能拿 ITA）。

5.3三线 · 工签先行 + PNP Support Letter

PR 等待期不空窗 · 全家先入境工作

对"国内已没工作 + 已拿省提名 + 不想在国内等 6 月" 的家庭，最实操的玩法：

  • 拿到省提名后立即向省 PNP 申 Work Permit Support Letter
  • 凭 Support Letter 向 IRCC 申 LMIA-exempt 工签（代码 A75 PNP / C50 trade agreement）· 1–3 年 · 配偶 OWP 派生
  • 全家持工签先入境 · 落地后等 PR 审理完成
  • PR 落地后所有工签自动结束 · 切换到 PR 身份

这把 "PNP 6–9 月 PR 等待"变成 "落地工作中等"，对子女教育衔接尤其重要。

§ 6家庭三线图 · 典型 5–6 年到入籍

Y0 准备 EE 档案 · CRS 自评 · IELTS · 学历 ECA Y0+1M 进 EE pool · 或同时投 BC PNP / OINP / AAIP profile Y0+3–6M收 NOI / BC ITA · 递省提名申请 Y0+6–12M 拿到 Nomination Certificate · EE +600 CRS Y0+9M （可选）申 Support Letter 工签 · 全家先入境 Y1 收 EE ITA · 60 天递 e-APR · 全家文件同批 Y1.5 COPR · 全家登陆 PR · 工签自动转 PR Y1.5–Y4.5 持 PR 累 3 年物理存在 Y4.5 递入籍 · 入籍考试 · 宣誓 Y5–Y6 拿加拿大护照 （中国国籍法 §9 自动失中国籍 · 注销中国护照 / 户籍）

注：上图假设"裸 PNP" 主线 + Y0+9M 起 Support Letter 工签辅线。若选 OINP Tech Draws 但等不到 NOI · 时间线可能拉长 2–3 年；最坏情况须切到 ca-gts / ca-ict 工签先行重走。

§ 7核心风险点

PNP 申请阶段的结构性风险

  • 2024 起 PNP 总配额砍半：各省 draw 频率与每次邀请数显著下降 · 2024-25 OINP Tech Draw 全年仅 1–6 次（vs 2022-23 每月）· cutoff 上移 30+ 分
  • NOC 清单浮动：BC PNP / OINP Tech NOC 每年由各省刷新 · 部分 NOC（如 22220 网络技术员）可能被剔除
  • 申请人 EE 档案过期：EE 档案有效期 12 个月 · 期间未拿 ITA 须重交 · 期间英语 / 学历认证证书也可能过期
  • "打算在该省居住"陈述：OINP / BC / AAIP 在提名审理与 PR 接收时各审一次 · 申请人若已被发现在另一省找过工作 / 看过房 / 持有 lease，提名可能被撤销
  • 跨省迁移道德风险：提名 1 年内跨省迁移可能在 PR 卡续期 / 入籍审理时被质询

Express Entry / PR 阶段的风险（与 ca-gts / ca-ict 共有）

  • PNP 通道 PR 处理慢：PNP-Enhanced PR 处理 6–9 月（vs CEC-only EE 6 月）· 不可加急
  • Medical / Criminal 背调：既往健康问题 / 其他国家刑事记录可能 RFE 甚至拒批
  • 家属同步：申请期间家属婚姻状态 / 健康状态变化 → 必须如实报告

PR 维持与入籍阶段（与 ca-gts / ca-ict 共有 · 北京特别提醒）

  • Residency Obligation：5 年内不足 730 天物理存在 → PR 可撤销 · 最常见坑：PR 登陆后被中国雇主重新雇佣长期回去
  • 跨境社保切割：中加无双边社保协定 · 详见 ca-ict § 2.4
  • 子女兵役 / 学籍同步：父母失中国籍后子女户籍 / 学籍需同步处理

§ 8备选切换 · 这条走不通怎么办

  • CA · GTS（Global Talent Stream）2 周工签 + 后续 EE
    门槛：雇主走 Cat A（referral）/ Cat B（14 NOC 清单）· 雇主 LMBP。何时考虑：PNP cutoff 跨不过 / 等不到 NOI · 但有加拿大雇主愿走 GTS。价值：2 周入境工作 · 满 1 年 CEC + 不依赖省提名 · GTS 工签期间也可叠加 PNP（双保险）。 详情：本站 · CA Global Talent Stream
  • CA · ICT（Intra-Company Transferee）跨国公司内部调动
    门槛：原雇主 ≥ 1 年 + Executive / Senior Manager / Specialized Knowledge 三档 · LMIA-exempt。何时考虑：已在 MSFT / Google / Amazon 等跨国公司工作 · 偏好"工签先行 + CEC 路径"。价值：免 LMIA · 配偶 OWP · 期间可叠加 PNP。 详情：本站 · CA ICT
  • CA · FSW（Federal Skilled Worker）纯海外申 EE 无 PNP
    门槛：海外 1 年 NOC 经验 · 67/100 初筛 · CLB 7。何时考虑：PNP 等不到 NOI 且 CRS 自评 ≥ 480 · 直接靠 general draw 拿 ITA。价值：不绑省 · 完全靠 CRS 打分。劣势：2025 general draw cutoff 通常 500+ · 中国大陆出生 30+ 岁普通申请人极难够到。 详情：ca-gts § 3 内 FSW 子节
  • CA · Start-Up Visa创业一步永居
    门槛：designated organization Letter of Support · 团队 ≤ 5 人。何时考虑：不打算靠雇主 + 有创业项目。 占位：本站 · CA Start-Up Visa（未建）· 官方：IRCC · Start-Up Visa
  • CA · 其他省 PNP（小省 SK / MB / NB / NS）本页未单独覆盖
    门槛：SINP / MPNP / NBPNP / NSNIP 各省自定。何时考虑：BC / ON / AB 三省 cutoff 高 + 你不挑大城市 · 愿意去 Saskatoon / Winnipeg / Halifax 等中小城市。价值：cutoff 普遍更低 + 一些省（如 SK Occupations In-Demand）甚至不需要 Job offer。劣势："在该省居住"要求更严 · 跨省迁移道德义务更高。 官方：各省 PNP 门户 · 本站未单独建路径页
  • UK · Skilled Worker英国雇主担保
    门槛：Sponsor licence 雇主 + 一般档 £41,700 / ISL £33,400。何时考虑：同时有英国 offer · 偏好伦敦薪资水位 + 英国护照。 详情：本站 · UK Skilled Worker
  • IE · CSEP爱尔兰 EU 替代
    门槛：CSOL 内岗位 + €38K / 非清单 €64K。价值：2 年 Stamp 4 + 5 年入籍 + EU 护照终局。 详情：本站 · IE CSEP
  • NZ · SMC新西兰一步直发居留
    门槛：6 分制三类组合。价值：5 年入籍 + NZ→AU SCV 跳板。 详情：本站 · NZ SMC
  • AU · National Innovation Visa澳大利亚一步永居
    门槛：国际公认成就 + AU 提名人 Form 1000。价值：4 年入籍。 详情：本站 · AU NIV
  • US · H-1B / L-1A / L-1B美国主流工签
    门槛：H-1B 抽签 · L-1 跨国 ≥ 1 年。何时考虑：偏好美国 TC。劣势：EB 排期对中国大陆出生申请人极长。 详情：本站 · US H-1B · L-1A · L-1B
