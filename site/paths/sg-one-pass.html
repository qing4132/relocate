0.3三大优势（为什么选这条）

5 年长签 · 解耦雇主 · 低税 + 亚洲枢纽

① 5 年长签：EP 首次最多 2 年，换工作还要重办；ONE Pass 一次 5 年免刷，中间换工作 / 跳雇主 / 自办公司都不需要重新申请。

② 完全解耦雇主：可同时为多个新加坡公司工作、担任多家公司的董事、创办自己的公司并聘自己——这是 EP 完全做不到的。对"一份高管正职 + 几个董事 + 自己 SPV" 的典型高净值职业形态几乎无缝。

③ 低税 + 亚洲枢纽：居民个税累进到 24%（SGD 100 万以上档，2024 后从 22% 调升），整体税负明显低于美 / 英 / 德；时区覆盖亚太全段，飞中国 / 香港 / 日本 / 印度都在 2–6 小时内。IRAS 的 tax residency 183 天规则简单可算。

§ 1ONE Pass 入境 · 门槛与申请

ONE Pass 由 MOM 审批、MOM 发准证。门槛分为两档：薪资档（硬指标）与成就档（软指标）。达任一档即可递。

1.1申请资格 · 两档门槛

§1.1 · MOM ONE Pass Eligibility
薪资档 · 月薪 ≥ SGD 30,000（最近 1 年）

"最近 1 年"指过去一年在新加坡现有雇主 或 海外知名企业获得的固定月薪（fixed monthly salary）。口径：

  • 固定工资（不含 bonus / RSU / 奖金 / 补贴）· 若一部分是 RSU / 股票，按年化现金等价方式折算
  • 若"海外雇主"：必须是市值 ≥ USD 500M 或年收入 ≥ USD 200M 的知名企业（MOM 依公开数据自主核实）
  • 若在新加坡就业：需已在新加坡持 EP ≥ 6 个月，且达到本档月薪

核心参考：月薪 SGD 30,000 大致等价于年薪 SGD 360,000 固定（不含 bonus），约 USD 270,000。

§1.1 · MOM ONE Pass Eligibility · Outstanding Achievements
成就档 · 国际公认成就（免薪资门槛）

不达薪资门槛但在 科学 · 科技 · 学术 / 研究 · 艺术文化 · 体育五个领域内有国际层面可佐证的杰出成就，亦可申。MOM 的表述是 "outstanding achievements in arts and culture, sports, science and technology, or academia and research"。实务接近美国 EB-1A 的证据结构：

  • 国家或国际级奖项（不是公司内部 / 地方奖）
  • 顶刊 / 顶会论文与显著引用、媒体报道
  • 主要机构的领导 / 关键角色、严格会员标准的专业团体成员（Fellow 等）

MOM 保留是否受理的完全裁量权——本档比薪资档模糊得多，通常需要配一份"国内国际背书" + 实际业务 / 研究 / 创业计划书。

1.3家属派生 · DP · LOC

§1.3 · MOM Dependant's Pass · Letter of Consent
配偶 + 21 岁以下未婚子女 → DP · 配偶凭 LOC 可工作

Dependant's Pass（DP）：

  • 谁可申：法律意义上的配偶（同性伴侣不纳入） · 21 岁以下未婚子女（含继子女与领养子女）
  • 有效期：随主申的 ONE Pass · 主申失效则 DP 同步失效
  • 居留权：与主申共处 · 无月天数要求但保留身份须保持非丢 DP

配偶工作权 · LOC（Letter of Consent）：配偶持 DP 后另申 LOC，批了即可在新加坡自由工作——不限雇主、不限行业、无薪资门槛。这是新加坡相对美国 H-4 / 日本 HSP 配偶就业权最宽的一处。

21 岁以上子女：DP 失效，须转 Long-Term Visit Pass（LTVP，父母赞助）或独立准证（EP / S-Pass / 学签）。

§ 2准证维持 · 多雇主 · 开公司 · 续签

ONE Pass 的最大价值是"持证期间无雇主绑定"——这带来全新的收入结构可能性，也意味着续签时要对自己的收入 / 成就账本负责。

2.1在持证期间可以做的事

  • 为多家新加坡公司同时工作：可同时是 A 公司 CTO、B 创业公司 advisor、C 投资公司独立董事——只要全部按 ONE Pass 的报工范围
  • 自办公司并担任 Director：可设立 Pte Ltd 聘自己（部分高净值"正职 + Holding + 咨询"结构的标准做法）
  • 就地换雇主：无须 MOM 报备审批，直接入职新公司；但换主雇主后的收入仍须在续签时计入累计
  • 出国短期出差：无最短居住天数硬指标，但 PR / 入籍阶段 ICA 仍会看居住比率

2.2续签门槛 · 三条路径满足一条

§2.2 · MOM ONE Pass Renewal
5 年末续签 · 再加 5 年

MOM 公布的续签口径是"三条中满足一条"：

  1. 累计收入：过去 5 年在新加坡的可评估收入（assessable income，以 IRAS 税单为准）累计 ≥ SGD 1,500,000；或
  2. 持续高薪：当前月薪仍 ≥ SGD 30,000 固定；或
  3. 持续成就：在 ONE Pass 持证期间为新加坡经济 / 创业生态创造实质贡献，MOM 以个案裁量。典型：
    • 在新加坡新设公司、雇佣本地员工（MOM 会看 Total Employment Manpower 数字）
    • 本地募资 / 产生 revenue
    • 持续公开演讲 / 研究 / 艺术产出并获本地媒体报道

口径会调整：2024 年定下的数字未来可能动，任何时候以 MOM 当期公告为准。

2.3税务 · IRAS

持证后在新加坡居住 ≥ 183 天（当年内）或跨 3 年累计 ≥ 183 天即为"tax resident"，适用累进税率（最高 24%，2024 后）。非居民按统一 24% 或最低税率中较高者计。

  • 海外收入：新加坡采属地征税——在新加坡境外赚取且未汇入新加坡的收入通常不征税（Not-Ordinarily-Resident / Non-Ordinarily-Resident 判定特别细，请咨询税务师）
  • 双重征税：中国与新加坡有 DTA · 美国无全面 DTA，但持 H-1B / 绿卡等美国税务连接仍须向 IRS 申报全球收入（LPR 路径兼容度差）
  • CPF（公积金）：外籍准证持有人不必缴、也不能缴 CPF；转 PR 后开始缴

§ 3升 PR · ICA 裁量通道 · PTS Scheme

新加坡 PR 由 ICA 按 PTS（Professionals / Technical Personnel and Skilled Workers）通道审批；持 ONE Pass 满 ≥ 6 个月起即可递，但批与不批完全靠 ICA 裁量——无配额 / 无公开打分表 / 拒批不给理由，一年一次或两次重递即可。

§3 · Immigration Act §10 · §12 · ICA PR e-Service
门槛是"不明示、靠综合印象"

ICA 审查的常见综合维度（业内总结、非官方公开）：

  • 长期价值感：在新加坡已经纳税多久 / 有多稳定的雇主 / 有没有本地雇员 / 有没有在新加坡买房或长租
  • 经济产出：IRAS 税单的"assessable income"数字比 offer letter 的名义工资更重要——ICA 看的是"你实际交了多少税"
  • 社会嵌入：子女读的是本地学校还是国际学校、配偶是否在新加坡就业 / 开公司、是否有本地社区 / NGO / 行业协会参与
  • 学历 · 行业 · 年龄：ONE Pass 持证本身是强信号（MOM 已筛过），ICA 通常对 ONE Pass 持证者的审批比 EP 更积极，但仍无保证

时间线：

  • 递 e-PR（在线表 + 证据扫描件）→ 4–6 个月出结果
  • 批准：IPA 信 → 3 个月内完成 formalities 并领 PR 卡（Entry Permit）
  • 拒批：不给理由 → 建议等 6–12 个月后（收入 / 职位 / 社会参与度有新变化时）重递

PR 的 Re-Entry Permit（REP）：PR 卡本身每 5 年随附 REP，用于离境后返回的权利；若 REP 过期时人在国外未及时续，PR 身份可被取消——这是维持 PR 的最重要硬性约束。长期不在新加坡的 PR（如回中国工作）容易在 REP 续期时被拒。

3.1配偶 / 子女随 PR

主申拿 PR 后，配偶与 21 岁以下未婚子女可同一份 e-PR 表附申；也可在主申获批后单独递。配偶获 PR 后即独立工作权（不再需要 LOC / DP），并开始缴 CPF。子女若在新加坡出生 + 父母至少一方为新加坡公民，自动公民；若父母为 PR，则子女可申 PR。

3.2CPF（Central Provident Fund）义务

转 PR 当月起雇主与雇员开始缴 CPF（基本是三个子账户 OA / SA / MA）· 缴费率首 1 年较低（14% 雇员 + 4% 雇主），从第 3 年起按新加坡公民口径（20% + 17%）。CPF 同时是强制养老金与住房基金（可用 OA 买政府组屋 BTO / 私宅 DP）——成为进入公民的重要财务节点。

§ 4升公民 · Singapore Citizenship · 单一国籍

新加坡实行单一国籍（Constitution Art 134）——入籍时必须放弃原国籍。对中国大陆出生者而言，中国《国籍法》§9 也同步生效："定居外国的中国公民，自愿加入或取得外国国籍的，自动丧失中国国籍"——两条冲突规则彼此自洽。

§4 · Constitution Part X · Art 122–141 · Citizenship by Registration
PR ≥ 2 年后可申 · 年满 21 · 无犯罪

法定门槛（Constitution Art 123 - Registration）：

  • 年满 21 岁；
  • 持 PR ≥ 2 年（实务通常建议 ≥ 5 年 PR 再递，批准率更高）；
  • 能够讲 英 / 马来 / 华 / 泰米尔任一官方语言；
  • 有意永久定居新加坡；
  • 品德良好、无重大犯罪记录。

实务门槛（ICA 综合裁量）：与 PR 同构——纳税多少、本地雇员、是否买 HDB / 私宅、社区嵌入、子女就学、配偶 / 孩子是否也递。

时间线：

  • 递 Citizenship e-Service → 6–12 个月出结果
  • 批准：邀请参加 Citizenship Journey 半日项目 + 宣誓（Oath of Renunciation, Allegiance and Loyalty）
  • 宣誓后：3 个月内须完成放弃原国籍手续 · 注销中国护照（若中国大陆出生）· 领新加坡身份证与护照

4.121 岁以下子女同期入籍

主申获批公民后，未成年子女（含 PR 子女 / DP 子女 / 随同申请中的子女）可依 Constitution Art 122 以 Citizenship by Descent 或 Registration 一同入籍，无须单独走 2 年 PR 期——这和美国 CCA 2000 的自动入籍规则功能上类似，但技术上是"同一申请下的合并处理"而不是"法律自动触发"。

4.2中国籍的同步处理

宣誓即生效的双向触发：

  • 新加坡侧：Constitution Art 134 要求 registered citizen 在合理时间内放弃原国籍并提交放弃证明（渣打 / 驻华使馆可代转交中国大使馆的《退籍申请》）
  • 中国侧：依中国《国籍法》§9，在宣誓加入新加坡国籍那一刻"自动丧失中国国籍"——《国籍法》§10 规定的"申请退籍"和 §9 的"自动丧失"并列生效，实务上建议尽早办理 §10 的正式退籍证明以便中国户籍注销与护照注销
  • 护照处理：中国驻新加坡大使馆会在确认失籍后配合完成护照注销、户籍注销；此后回中国须持新加坡护照 + 签证

不是所有人都能走到这一步。ICA 拒批率公开讨论中估计在 50% 以上；重大分歧点通常是"是否足够嵌入新加坡社会"——在新加坡住但核心业务仍在海外的人容易栽在这一条。

§ 5家庭玩法 · 三种典型配置

ONE Pass 家庭相对 EP 家庭的最大优势在"配偶 LOC 无门槛工作权"与"5 年长签"——这让家庭可以在更从容的节奏下做选择。以下三种是最常见的配置。

5.1双线 A · 主 ONE Pass + 配偶 LOC

最常见组合 · 配偶凭 DP + LOC 自由工作
  • 触发时机：主申拿 ONE Pass 即可（不用等）
  • 配偶方：DP 发卡后立即递 LOC → 4–6 周批 → 任何雇主 / 任何行业就业或开公司。LOC 没有月薪门槛、没有行业限制（安保 / 医生等个别需进一步注册的职业除外）
  • PR 时机：建议主申持 ONE Pass 12–18 个月后主+副同期递 PR，批准率比单递高

5.2双线 B · 主 ONE Pass + 配偶独立 ONE Pass / EP

夫妻双方都高薪或双方都有 ONE Pass 资格时最稳健
  • 触发时机：配偶本人月薪达 SGD 30k 或本身有国际成就
  • 优势：两条独立准证 · 两条独立 PR 线 · 两条独立公民线 · 家庭整体韧性最高；配偶失去 ONE Pass 资格时可用 EP 无缝接续
  • PR 实务：两人各自独立递 e-PR，通常有利于 ICA 印象

5.3双线 C · 主 ONE Pass + 配偶 EP

若配偶薪资未达 ONE Pass 门槛但找到愿意办 EP 的雇主，也可走 EP 独立线。EP 绑定雇主 + 首次 2 年，相对不稳定，但适合配偶薪资 SGD 5.6k–30k 的档位。

§ 6家庭三线图 · 典型 3 年 PR + 2 年公民节奏

以"主申 ONE Pass + 配偶 LOC + 1 个学龄子女"为例，最快节奏的时间线大致如下。实际视 ICA 裁量与个人情况浮动。

Y0 主申 ONE Pass 下签 · 全家入境 Y0+1M 配偶 LOC 生效 · 开始工作或开公司 Y1 子女适应本地学校 · 开始纳本地税 Y2 税单累计清晰 · 全家同期递 PR Y2.5 PR 批 · 开始缴 CPF · 买 HDB / 私宅的节点 Y4–Y5 PR 满 2 年 · 开始看公民节奏 Y5–Y6 递公民 · 建议 ≥ 5 年 PR 再递以提升批准率 Y6–Y7 公民批 · 宣誓 · 放弃中国籍 · 注销中国护照 · 领新加坡 IC

相比之下，EP 路径通常 Y3–Y4 递 PR（前期税单更薄），总时长约 Y8–Y10 入籍；ONE Pass 的 5 年长签与跳雇主自由让前 2 年的"税单厚度"可以更快建立。

§ 7核心风险点

ONE Pass 持证期的结构性风险

  • 续签口径的不确定性：累计 SGD 1.5M / 5 年的硬指标 2024 年刚定，未来可能上调或加新口径。规划时建议按 "SGD 1.5M + 持续高薪" 两条都达到冗余。
  • 收入结构的税务镜像：一部分高净值持证人把收入留在海外控股公司不汇入新加坡 → IRAS 税单"薄"→ ICA PR / 公民审批时不利。避险：主动提高本地 assessable income 比例。
  • 家庭 LOC 失效：主申失去 ONE Pass 资格（如续签被拒），配偶 DP + LOC 同步失效，整个家庭进入倒计时。缓冲：配偶独立 path（双线 B）。

PR / 公民阶段的裁量风险

  • 无公开打分：不像加拿大 Express Entry 有 CRS，新加坡 PR / 公民都没有公开打分表；ICA 可以不给理由拒批，这是结构性信息不对称。
  • 长期离境风险：PR 持有人若长期在国外（如回中国工作）→ REP 续期易被拒 → PR 失效。规划回亚洲频繁出差的工作模式时须注意物理存在比例。
  • 子女兵役：PR 男性子女 16.5 岁起须登记服兵役（NS），18 岁起入伍服役 2 年。若 15 岁前未拿 PR，入 PR 年龄越晚、NS 豁免空间越小——若放弃 PR 以回避 NS，将影响父母的 PR / 公民状态 + 未来入境资格。
  • 宣誓后的 3 个月：公民宣誓后 3 个月内须完成原国籍放弃——中国领馆实际办理周期可能超过 3 个月，需提前安排。

并发美 / 日 path 的互斥

  • 与美国 LPR 的冲突：若同时持有美国绿卡，Reentry Permit + 全球纳税义务与新加坡 PR / 公民路径互斥——规划时需在某个阶段二选一。
  • 与日本永住的冲突：日本要求"永住后以日本为生活中心"，与新加坡 PR 的居住比率要求同时存在时须做家庭分居 / 时序切换规划。

§ 8备选切换 · 这条走不通怎么办

ONE Pass 门槛高（月薪 30k 或国际成就）· ICA 裁量重。下面按降档次序列出备选与跨国替代。

  • SG · EP（Employment Pass）最常见的降档
    门槛：月薪 ≥ SGD 5,600（金融 6,200）· 依年龄阶梯升档 · COMPASS 打分制（工资 + 学历 + 多元化 + 本地雇员占比）。何时考虑：月薪达不到 SGD 30,000 但有 offer。价值：仍可走 DP / LOC · PR / 公民通道相同 · 但续签 + 换工作更繁琐。 官方：MOM · Employment Pass
  • SG · Tech.Pass2023 后逐步并入 ONE Pass
    门槛：月薪 ≥ SGD 20,000（历史档）或有海外科技公司 5–10 年高管经验。何时考虑：处于"薪资差一点 + 科技背景强"的尴尬区间。价值：2 年长签 · 可多雇主 · 续签看"5 项 KPI 中 4 项"。新申请者 MOM 一般建议直接考虑 ONE Pass；已持 Tech.Pass 可继续。 官方：EDB · Tech.Pass（EDB 站反爬 · 本地未归档）
  • SG · PEP（Personalised Employment Pass）解耦雇主的老款
    门槛：最近年薪 ≥ SGD 270,000（2023 年从 144k 大幅上调）。何时考虑：ONE Pass 成就档证据不足但年薪高。价值：3 年不可续 · 不绑定雇主 · 失业后 6 月宽限期。门槛与 ONE Pass 薪资档接近但年限更短、不可续——基本被 ONE Pass 覆盖，仅余历史持有人。 官方：MOM · Personalised Employment Pass
  • US · EB-1A杰出人才自荐绿卡
    门槛：一次性国际大奖 或 8 CFR 204.5(h) 10 条标准中 ≥ 3 条 + Kazarian 两段论整体判定。何时考虑：申请人本身在科研 / 工程 / 创业领域具备国际显性成就，且偏好美国。价值：自荐 · 免 PERM · 大陆出生排期 ≤ 2 年；配偶与子女同获绿卡。 详情：本站 · 美国 EB-1A 杰出人才自荐绿卡
  • US · O-1A杰出人才签证 · 非移民
    门槛：8 CFR 214.2(o)(3)(iii) 八项标准中至少三项。何时考虑：还不够 EB-1A、但需立即入境美国工作；或把 SG 作为备胎、主 path 放美国。价值：无年度配额 · 3+1+1…无限续签 · 常为 EB-1A 热身。 详情：本站 · 美国 O-1 杰出人才
  • JP · HSP 1 号イ / ロ / ハ高度専門職 · 永住最短 1 年
    门槛：法务省积分制 ≥ 70 分（年收入 + 学历 + 年龄 + 研究成果 + 经营业绩）。何时考虑：偏好日本市场、薪资档位介于新加坡 EP 与 ONE Pass 之间。价值：80 分可 1 年即申永住；配偶可全职。 详情：本站 · 日本 HSP 1 号ロ 技术系
  • UK · Global Talent科技 / 学术 / 艺术三赛道背书
    门槛：经 Tech Nation（已退役，由 RAEng 等接管） / Royal Society / British Academy 等指定机构 endorsement。价值：无雇主绑定 · 5 年可申 ILR；Exceptional Talent 档 3 年可申 ILR。 详情：本站 · UK Global Talent（到入籍） · 官方：GOV.UK · Global Talent visa
  • CA · Global Talent Stream雇主主导 · 2 周处理
    门槛：进入 ESDC 指定职业清单，雇主提交 LMBP（Labour Market Benefits Plan）。价值：2 周快速处理；配偶自动 open work permit；接续 Express Entry → PR 通道。 占位：本站 · CA Global Talent Stream（未建）· 官方：ESDC · Global Talent Stream
  • AE · Golden Visa10 年居留 · 低税
    门槛：专业人才档（月薪 AED 30,000+）或投资档（AED 2M+ 房产 / 基金）或特殊成就档。价值：10 年长签 · 零个人所得税 · 配偶与子女一同。劣势：不提供公民化通道、长期定居须重复续签。 占位：本站 · UAE Golden Visa（未建）· 官方：UAE ICA · Golden Visa
